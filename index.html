nodebrewのインストール

nodebrewについて

  • github.com/hokaccha/nodebrew
  • Node.js version manager
  • perlbrew/plenv + perl-build, rvm/rbenv + ruby-buildみたいなもの
  • OS X, Linux向け
  • Perl製
  • 類似ツールにnvm, nave, nなどがある
  • 参考：node.js の環境管理ツール nodebrew - Block Rockin' Codes

nodebrewを使わないインストール方法

  • nodejs.org/downloadの
    • インストーラを使ってインストール
    • バイナリのアーカイブをダウンロードして展開する
    • ソースコードからコンパイル

nodebrewをインストール / 1

OS X

$ curl -L git.io/nodebrew | perl - setup

Ubuntu, etc.

$ wget -qO - git.io/nodebrew | perl - setup

nodebrewをインストール / 2

OS X

$ echo 'export PATH=$HOME/.nodebrew/current/bin:$PATH' >> \
~/.bash_profile
$ source ~/.bash_profile

Ubuntu, etc.

$ echo 'export PATH=$HOME/.nodebrew/current/bin:$PATH' >> \
~/.bashrc
$ source ~/.bashrc

node.jsとnpmをインストール / 1

バージョンの確認

$ nodebrew ls-remote

0.10系（安定版）の最新バージョンをインストール

$ nodebrew install-binary 0.10

node.jsとnpmをインストール / 2

インストールされたnode.jsを使うように指定

$ nodebrew use 0.10

node.jsとnpmが使えるか確認

$ node --version
v0.10.12
$ npm --version
1.2.32

node.jsを使ってスクリプトを実行する

ワンライナー

Hello, World!

$ node -e 'console.log("Hello, World!")'
Hello, World!

REPL / 1

対話的に実行する

$ node

ヘルプの表示

> .help
.break  Sometimes you get stuck, this gets you out
.clear  Alias for .break
.exit   Exit the repl
.help   Show repl options
.load   Load JS from a file into the REPL session
.save   Save all evaluated commands in this REPL session to a file

REPL / 2

なにか実行してみる

> NaN === NaN
false
> isNaN(NaN)
true

終了

> .exit
$

スクリプト

何かスクリプトを書いて実行する

// index.js
// "Hello,"と実行時引数を標準出力に出力
console.log("Hello,", process.argv[2]);

nodeのあとにスクリプトを渡して実行する

$ node index.js node.js!
Hello, node.js!

npmを使ってモジュールをインストール

request

HTTPリクエストが簡単に書けるモジュール

$ npm install request
var request = require('request');
request('http://www.google.com', function (error, response, body) {
  if (!error && response.statusCode == 200) {
    console.log(body);
  }
});
$ node index.js

Bower

Twitter謹製フロントエンドパッケージマネージャ

$ npm install -g bower
$ bower install jquery underscore

これでjQueryとunderscore.jsがbower_components/にダウンロードされる

そのほか、便利なモジュール

Grunt
タスクランナー、Rakeみたいなもの
grunt-contrib-watchと各種grunt-contribを組み合わせると自動コンパイルできる
Express
SinatraにインスパイヤされたWebフレームワーク
Socket.IO
WebSocketサーバを作る際に便利なモジュール
node-dev
ファイルを保存したらファイル読み込み直して再実行する
RubyのShotgunとかSinatra/Reloaderみたいなもの
node-inspector
GoogleChromeのデベロッパーツールでnode.jsをデバッグできるようにするモジュール
