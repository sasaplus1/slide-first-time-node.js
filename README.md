# flowtime-slide-template

flowtime.js slide template

## start grunt tasks

    $ npm install
    $ npm start

start task of jade, stylus and coffee.

start watch's task when end these tasks.
this task is compile files if target files modified.

## appended keys

- Enter
  - move to next page
  - similar to down key
- Shift-Enter
  - move to previous page
  - similar to up key
